Manual

- Double-click for detail map

- Single click adds a new point to the region

- Clicking on a point opens a dialog box to edit or delete a point


- The icon 'Download' button leads to download a JSON file with data points

- To save the new points replace the downloaded JSON file in the directory: '.../maps/data'


- To change the path to the map files, make changes in the file '.../js/interactive-maps.js' string: pathMaps: 'maps/'
